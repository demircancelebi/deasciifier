# Türkçe Deasciifier

Deasciifier yabancı metinleri Türkçeye çevirmenize yarayan bir uygulamadır.

## Gereksinimler

- Mac OS X
- node / npm

## Yükeleme

```bash
npm install deasciifier -g
```
